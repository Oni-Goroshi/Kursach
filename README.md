# 🔐 Журнал паролей — Password Journal

> Учебный проект: локальный менеджер паролей с шифрованием алгоритмом RSA  
> Кубанский государственный университет · Курсовая работа · 2025–2026

---

## 📋 О проекте

**Журнал паролей** — десктопное приложение на **C++ / Qt 6**, которое позволяет безопасно хранить пароли на компьютере. Все пароли шифруются алгоритмом RSA перед записью в файл — в открытом виде они нигде не сохраняются.

### Зачем это нужно?
По статистике, 93% пользователей не ведут защищённый журнал паролей. Это приложение — первый шаг к цифровой гигиене.

## 🧮 Алгоритм шифрования — RSA

Используется **RSA** (Rivest–Shamir–Adleman) — алгоритм асимметричного шифрования с открытым ключом.

### Генерация ключей

Из мастер-пароля детерминированно вычисляются два простых числа **p** и **q**:

```
n   = p × q                     — модуль
φ   = (p−1) × (q−1)             — функция Эйлера
e   — открытая экспонента: НОД(e, φ) = 1
d   — закрытая экспонента: d × e ≡ 1 (mod φ)

Открытый ключ:  (e, n)
Закрытый ключ:  (d, n)
```

### Шифрование символа

```
C = m^e mod n
```

где `m` — числовой код символа (Unicode), `C` — шифртекст.

### Дешифрование

```
m = C^d mod n
```

### Пример (мастер-пароль `hello`)

Из пароля выбираются `p = 61`, `q = 53` → `n = 3233`, `φ = 3120`, `e = 17`, `d = 2753`

| Символ | Unicode m | Шифртекст C = m^17 mod 3233 |
|---|---|---|
| `p` | 112 | 2726 |
| `a` | 97 | 1444 |
| `s` | 115 | 1986 |

В файл записывается строка вида `2726,1444,1986,1986` — без мастер-пароля расшифровать невозможно.

> ⚠️ **Учебная реализация:** используются малые простые числа для наглядности. В промышленных системах RSA применяется с ключами от 2048 бит.

## 🚀 Сборка и запуск

### Требования
- Qt 6.x — [скачать установщик](https://www.qt.io/download-qt-installer)
- Qt Creator (входит в установщик)
- Компилятор: MinGW (Windows) или GCC/Clang (Linux/macOS)

### Через Qt Creator (рекомендуется)
1. Установите Qt 6
2. Откройте Qt Creator → `Файл → Открыть файл или проект`
3. Выберите `PasswordJournal.pro`
4. Нажмите **▶ Запустить** (Ctrl+R)

### Через командную строку
```bash
git clone https://github.com/ВАШ_НИК/password-journal.git
cd password-journal
qmake PasswordJournal.pro
make               # Linux/macOS
mingw32-make       # Windows
./password_journal
```

## 📖 Как пользоваться

1. **Запустите** → введите мастер-пароль (НЕ сохраняется)
2. **Добавьте запись:** `➕ Добавить` → заполните поля
3. **Сохраните:** `💾 Сохранить`
4. **При следующем запуске** введите тот же мастер-пароль

> ⚠️ Забытый мастер-пароль = потеря доступа к данным!
